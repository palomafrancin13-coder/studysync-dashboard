Este projeto consiste em um dashboard simples de gerenciamento de tarefas, desenvolvido com HTML e JavaScript. A aplicação permite ao usuário criar atividades contendo informações como título, descrição, data de entrega, status e prioridade, além de exibir todas as tarefas em uma lista dinâmica e organizada na interface.

A estrutura do projeto foi construída utilizando HTML para organização dos elementos da página e JavaScript para manipulação da lógica da aplicação, como criação e exibição das tarefas em tempo real.

A interface foi estilizada utilizando Tailwind CSS, um framework baseado em classes utilitárias que permite aplicar estilos diretamente no HTML de forma rápida e eficiente. Com isso, não foi necessário o uso de um arquivo CSS tradicional (style.css), pois toda a estilização foi feita por meio das classes fornecidas pelo Tailwind, garantindo um design moderno e responsivo.

O principal objetivo do projeto é desenvolver uma aplicação funcional para organização de tarefas, ao mesmo tempo em que são praticados conceitos fundamentais de desenvolvimento frontend, como manipulação do DOM, estruturação de interfaces e uso de frameworks de estilização.

Além disso, o projeto conta com integração com o Supabase para persistência de dados, permitindo que as tarefas possam ser armazenadas em um banco de dados. O script SQL utilizado para criação da estrutura do banco está disponível no arquivo database.sql no repositório.
